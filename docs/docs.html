Installation

Download Cream from the Downloads page. No extra dependencies required.

# Windows
cream-setup.exe

# Linux / macOS
chmod +x cream
./cream yourfile.cream

Hello World

print("Hello, World!")

Variables

let name = "Cream"
let version = 1
let ready = true

Types

let s = "hello"    # String
let x = 3.14      # Number
let ok = true     # Boolean
let nums = [1,2]  # List

Control Flow

if x > 5 { print("big") } else { print("small") }

while x > 0 { x = x - 1 }

for i in [1, 2, 3] { print(i) }

Functions

func add(a, b) {
  return a + b
}
print(add(3, 4))  # 7

Lists

let nums = [1, 2, 3]
nums.push(4)
print(nums[0])      # 1
print(nums.len())   # 4

Standard Library

print(x)           # output
input("prompt")    # user input
len(x)             # length
type(x)            # type name
range(1, 10)        # number range
random(1, 100)     # random int
floor(x) ceil(x) abs(x)

Cream IDE

A built-in PyQt6 desktop IDE with syntax highlighting, file tree, and integrated runner.

VS Code Extension

ext install MauyaApps.cream-language

Syntax highlighting, snippets, and .cream file association.
